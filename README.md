# rust_snippets

Коллекция полезных Rust-сниппетов для повседневного использования.

## Что это?

Rust-файлы, демонстрирующие различные темы: асинхронность, работа с файлами, JSON, сети, многопоточность, CLI, конфиги, логгирование, тестирование, обработка ошибок, паттерны и утилиты.

Каждый сниппет:
- Самодостаточен (можно проверить через `cargo check`).
- Содержит комментарии с пояснениями.
- Имеет название вида `snippet_<тема>_<timestamp>.rs`.

## Как использовать

1. Склонировать репозиторий.
2. Добавить зависимости в `Cargo.toml` (указаны в комментариях каждого файла).
3. Запустить конкретный сниппет: `cargo run --bin <имя_файла.rs>` или `cargo check`.

## Contributions

Сниппеты добавляются автоматически через heartbeat системы OpenClaw. Каждый heartbeat может добавить новый пример по выбранной теме.
